i. Uraian Materi Pembelajaran
    Pendidikan Pancasila di perguruan tinggi sangat penting untuk diajarkan kepada mahasiswa untuk memperkuat identitas nasional dan persatuan di tengah keberagaman yang ada. Tanpa pemahaman yang mendalam tentang sejarah dan nilai-nilai dasar bangsa, mahasiswa rentan terhadap paham yang bisa merusak eksistensi bangsa. Misalnya saja seperti kejatuhan negara besar seperti Uni Soviet ataupun juga Kerajaan Majapahit yang terjadi karena mereka tidak mengaktualisasi nilai nilai dasar yang ada dalam kehidupan berbangsa dan bernegara. Hal ini menegaskan kepada kita tentang pentingnya ideologi yang kuat dalam mencegah disintegrasi bangsa.

    Sejarah adalah guru bagi bangsa, dan pancasila mengajarkan kepada kita pembelajaran dari kesalahan di masa lalu untuk kelangsungan hidup bangsa sekarang dan juga kedepannya. Pancasila mewujudkan kecerdasan intelektual, emosional, spiritual, dan ideologis, yang mengarahkan bangsa menuju kemakmuran dan persatuan.

    Nilai-nilai inti pancasila, seperti gotong royong dan keadilan sosial, bukan hanya slogan, tetapi menjadi semacam panduan moral bagi kita dalam menghadapi tantangan sosial dan ekonomi modern. Di dunia yang semakin global ini, mahasiswa perlu pegangan yang kokoh agar tetap memiliki komitmen terhadap bangsa.

    #include <stdio.h>
#include <stdlib.h>

void classify_number(int N) {
    int length = snprintf(NULL, 0, "%d", abs(N)); // Menghitung panjang angka
    
    if (length == 1) {
        printf("Satuan\n");
    } else if (length == 2) {
        printf("Puluhan\n");
    } else if (length == 3) {
        printf("Ratusan\n");
    } else if (length == 4) {
        printf("Ribuan\n");
    } else if (length == 5) {
        printf("Puluh Ribuan\n");
    } else {
        printf("Bilangan terlalu besar\n");
    }
}

int main() {
    int N;
    printf("Masukkan bilangan: ");
    scanf("%d", &N);
    classify_number(N);
    return 0;
}
Selain itu, pancasila merupakan sumber dari segala sumber hukum, sehingga mahasiswa yang memahami konsep ini pastinya akan lebih kritis terhadap arah kebijakan pemerintah. Meskipun menghadapi beberapa tantangan, kini pancasila tetap menjadi kerangka kerja penting dalam pemerintahan Indonesia yang memastikan persatuan dan kesejahteraan bangsa.
